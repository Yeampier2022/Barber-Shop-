## Booking Flow

- Book a haircut with name, phone number, service, date, and time.
- The form saves the appointment to Firebase Firestore.
- The screen shows upcoming bookings.

## Run the app

``5`bash
npm install
npm start
```

> If `npm install` fails, try updating npm or use `npm install --legacy-peer-deps`.
